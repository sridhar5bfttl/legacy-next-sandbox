### Analysis Guidelines:
1. Identify all variables and their native data types.
2. Catalog all control loops, procedures, paragraphs, or event-driven functions.
3. Call out all external operations: file reads, database statements (`EXEC SQL`), API declarations (`Declare Lib`), UI dialog forms, or screen maps (`SEND MAP`).
4. Output ONLY the JSON block. Do not wrap it in markdown or add conversational preamble.
